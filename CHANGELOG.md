# Changelog

## 0.0.1

- Scaffolded VS Code extension with TypeScript and Vitest test suite.
- Added settings schema for enablement, cooldown, dedupe, audio source, and terminal patterns.
- Implemented detector wiring through a central failure event coordinator.
- Added Testing API, terminal, and task-based failure detectors.
- Implemented cooldown and dedupe policy engine.
- Added system audio playback with bundled and custom path resolution.
- Added output channel logging and manual sound test command.
- Documented usage, configuration, and runtime limitations.
